Отчёт по лабораторной работе №1 по ООП.

Описание действий, сделанных мной до первого commit.
1.Из проверенного источника была скачана система контроля версий git.
2.Далее она была настроена при помощи данных команд:
git config —global user.name "Your Name"
git config —global user.email "your_email@whatever.com"
git config —global core.autocrlf true
git config —global core.safecrlf warn
git config —global core.quotepath off
3.Далее был создал каталог с название OOP, в котором и будет происходить дальнейшее выполнение лабораторной работы.
4.Далее была произведена инициализация Git в данной папке при помощи пары команд:
cd D:\ООП - Переход в нужную папку.
git init - Инициализация выбранной папки.
5. Создаём текстовый файл для отчёта при помощи данной команды:
touch BelyanskiiArtem_PI-92_LR1.txt
Далее мы его индексируем:
git add BelyanskiiArtem_PI-92_LR1.txt
6. Создаём первый commit:
git commit -m "File Creation"
7. Создаём комит, в котором расказывается о проделанной работе до данного времени.
8. Создаём файл qw.txt, что бы продемонстрировать игнорирование. При помощи команды echo qw.txt > .gitignore создаю папку .gitignore и помещаем туда файл qw.txt. 
Добавляем файл .gitignore при помощи команды git add .gitignore и создаём commit с описанием выполненого действия.
9. Для того, что бы показать сравнение была использована команда git diff хеш1 хеш2.
10. Что бы показать удаление файла мы создаём файл udali_menya.txt, заносим его в репозитории при помощи команды git add.
Далее, при помощи команды git rm -f udali_menya.txt удаляем его.
